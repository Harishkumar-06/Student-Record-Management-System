



# Student Record Management System

# Overview
A menu-driven Student Record Management System developed in C using Singly Linked Lists and File Handling.

# Features
- Add Student Record
- Delete by Roll Number
- Delete by Name
- Modify Student Details
- Display Records
- Sort by Name
- Sort by Percentage
- Reverse Records
- Delete All Records
- Save Records to CSV
- Load Records from CSV

# Technologies Used
- C Programming
- Singly Linked List
- Dynamic Memory Allocation
- File Handling
- Makefile

make
